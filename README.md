# Framework Test

## Stack

- [React.Js](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org)
- [Styled components](https://styled-components.com/)
- [Storybook](https://storybook.js.org/)
- [Jest](https://jestjs.io)
- [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/)
- [Plop](https://plopjs.com/)

## Install and start the project

```bash
yarn
yarn start
```

## Components

### Creation

On component creation, i've implemented **plop**, that way the components will always be created following the same template to keep the code consistent.

**Component creation command:**

```
yarn generate or yarn generate <component name>
```

### Documentation

For component development and documentation we use the **storybook** tool, in the storybook we have access to all components ever created on the project.

**Storybook command:**

```
yarn storybook
```
